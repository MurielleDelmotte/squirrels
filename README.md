The goal of `{squirrels}` is to analyse the populations of squirrels of
Central Park, NYC.

The data is from the NYC Squirrel Census - raw data at NY Data portal.
Data were collected in Central park in October, 2018.

## Installation

You can install `{squirrels}` with:

``` r
# install.packages("remotes")
remotes::install_local(path = "~/squirrels_0.0.0.9000.tar.gz") 
```
